# 🤖 AURIX AI - Advanced Intelligent Voice Assistant

AURIX AI is a Python-based AI Voice Assistant that can understand voice commands, respond intelligently, and perform real-world tasks like reminders, web search, system control, and AI chatbot responses.

---

## 🚀 Features

- 🎤 Voice Recognition
- 🧠 NLP Intent Detection
- 💬 AI Chatbot System
- 📚 Wikipedia Search
- 🌤 Weather Information
- ⏰ Reminder System
- 💻 Open Applications
- 🌐 Web Search
- 😊 Sentiment Detection
- 🔊 Text-to-Speech AI
- 🧠 Memory System

---

## 🛠 Tech Stack

- Python
- SpeechRecognition
- pyttsx3
- Wikipedia API
- OpenWeather API
- TextBlob
- PyWhatKit
