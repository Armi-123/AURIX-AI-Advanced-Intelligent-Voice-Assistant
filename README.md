AURIX AI is a Python-based intelligent voice assistant designed to perform everyday tasks through voice commands. It combines **Speech Recognition**, **Natural Language Processing (NLP)**, **Text-to-Speech**, and **API integrations** to create an interactive desktop assistant capable of answering questions, managing reminders, searching information, launching applications, and automating common tasks.

This project demonstrates the implementation of a modular AI assistant architecture similar to modern virtual assistants such as Alexa and Google Assistant while focusing on Python development and practical AI concepts.
